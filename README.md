# sila's pacpac world

This repository will contain the full source for "sila's pacpac world" — a polished Pac‑Man style game with both a native Python/Pygame desktop build and a web (HTML5) build playable in Chrome/ChromeOS.

This commit is the initial scaffold. The pacpac-scaffold branch will contain the full project and a PR will be opened for review.

Planned deliverables:
- Python / Pygame desktop game (packaging via PyInstaller)
- HTML5 web port (Phaser) deployed to GitHub Pages for Chrome/ChromeOS play
- Smooth movement, improved ghost AI, menus, multiple levels
- Pink Pac‑Man + confetti on win; sad emoji effects on loss
- MIT license for code and assets
- GitHub Actions workflow to build native and web artifacts and deploy web build

See the pacpac-scaffold branch for the working project and PR.
